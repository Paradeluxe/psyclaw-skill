# stroop_experiment

Color-word Stroop task — 30 trials

## Overview

PsyClaw-generated PsychoPy experiment.

- `stroop_experiment.psyexp` — main PsychoPy Builder file
- `spreadsheets/` — conditions Excel files
- `assets/` — image/audio/video stimuli
- `run.sh` / `run.bat` — launch scripts

## How to run

1. Open PsychoPy (2023.2+)
2. File → Open → `stroop_experiment.psyexp`
3. Click green Run button

## Conditions

30 trials, randomized.

## Files

- `spreadsheets/stroop_conditions.xlsx` — per-loop conditions
- `assets/(none)` — stimuli
