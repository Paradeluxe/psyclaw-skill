# 菜单栏垂直居中修复

## 问题

`.project-menu-left` 在 `.project-menu-bar` 中没有垂直居中，内容贴着顶部边缘。

## 原因

`.project-menu-dropdown` 作为 `.project-menu-bar` 的 DOM 子元素，虽然设置了 `position: fixed`，但仍作为 flex 子项参与布局计算，干扰了 `align-items: center` 的效果。

## 修复方案

### 1. 垂直居中

给 `.project-menu-left` 添加强制位移：

```css
.project-menu-left {
    position: relative;
    top: 50%;
    transform: translateY(-50%);
}
```

### 2. 水平居中

```css
.project-menu-bar {
    justify-content: center;
}

.project-menu-left {
    width: fit-content;
    margin: 0 auto;
}
```

### 3. 防止 dropdown 干扰

```css
.project-menu-dropdown {
    flex: none;
    height: 0;
}
```

## 菜单栏阴影层级

将 chatbot 面板置于菜单栏之下：

```css
.project-menu-bar {
    z-index: 999999;
}

.chatbot-panel {
    top: 54px;      /* 从 48px 改为 54px，让出 6px 显示阴影 */
    height: calc(100vh - 54px);
    z-index: 999;   /* 从 1000 降到 999 */
}
```
