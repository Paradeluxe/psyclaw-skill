# 折叠面板间距控制方法

## 问题描述
在 "Outlining completed" 和 "Flowchart completed" 两个折叠面板之间需要控制间距，截图中红框标注的区域即为需要控制的间距部分。

## 控制方法

### 1. CSS 类名控制（推荐）

```css
/* 方法1：使用 margin 控制间距 */
.collapse-item {
  margin-bottom: 16px;  /* 控制每个折叠面板底部的间距 */
}

.collapse-item:last-child {
  margin-bottom: 0;  /* 最后一个不需要底部间距 */
}

/* 方法2：使用 gap 控制（父容器使用 flex/grid） */
.collapse-container {
  display: flex;
  flex-direction: column;
  gap: 16px;  /* 控制子元素之间的间距 */
}

/* 方法3：使用 padding 控制 */
.collapse-item + .collapse-item {
  padding-top: 16px;  /* 相邻的折叠面板之间添加顶部内边距 */
}
```

### 2. 组件属性控制（根据UI框架不同）

#### Ant Design
```jsx
<Collapse 
  className="custom-collapse"
  items={[...]}
/>

// CSS
.custom-collapse .ant-collapse-item {
  margin-bottom: 16px;
}
```

#### Element Plus
```vue
<el-collapse class="custom-collapse">
  <el-collapse-item>...</el-collapse-item>
</el-collapse>

<style>
.custom-collapse .el-collapse-item {
  margin-bottom: 16px;
}
</style>
```

#### 原生实现
```html
<div class="collapse-list">
  <div class="collapse-item">Outlining completed</div>
  <div class="collapse-item">Flowchart completed</div>
</div>

<style>
.collapse-list {
  display: flex;
  flex-direction: column;
  gap: 16px;
}

.collapse-item {
  border: 1px solid #e0e0e0;
  border-radius: 8px;
  padding: 12px 16px;
}
</style>
```

### 3. 间距变量规范

建议定义统一的间距变量：

```css
:root {
  --spacing-xs: 4px;
  --spacing-sm: 8px;
  --spacing-md: 12px;
  --spacing-lg: 16px;
  --spacing-xl: 24px;
  --spacing-xxl: 32px;
}

.collapse-item {
  margin-bottom: var(--spacing-lg);  /* 16px */
}
```

### 4. 响应式间距

```css
.collapse-item {
  margin-bottom: 12px;
}

@media (min-width: 768px) {
  .collapse-item {
    margin-bottom: 16px;
  }
}

@media (min-width: 1200px) {
  .collapse-item {
    margin-bottom: 24px;
  }
}
```

### 5. 特殊情况处理

#### 折叠状态不同的间距
```css
.collapse-item.is-expanded {
  margin-bottom: 24px;  /* 展开时更大的间距 */
}

.collapse-item.is-collapsed {
  margin-bottom: 12px;  /* 折叠时较小的间距 */
}
```

#### 分组间距
```css
.collapse-group {
  margin-bottom: 32px;  /* 组与组之间的间距 */
}

.collapse-group .collapse-item {
  margin-bottom: 8px;   /* 组内项之间的间距 */
}
```

## 最佳实践

1. **使用 CSS 变量**：便于统一维护和主题切换
2. **避免使用固定像素值**：考虑使用 rem 或 CSS 变量
3. **考虑相邻兄弟选择器**：`.item + .item` 可以精确控制相邻元素间距
4. **注意最后一个元素**：通常最后一个元素不需要底部间距
5. **保持一致性**：整个应用使用统一的间距规范
