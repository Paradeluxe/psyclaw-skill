## 设计系统核心特点

### 1. 视觉主题

MiniMax 的设计语言以白色为主，营造轻盈、通透的氛围：

- **主背景**: 纯白色 (`#ffffff`)
- **设计风格**: 类似 Apple 产品营销的清晰度 + 圆润友好的设计语言
- **视觉焦点**: 彩色产品卡片和 AI 模型插图

### 2. 字体系统

采用多字体组合策略：

| 字体 | 用途 | 回退字体 |
|------|------|---------|
| DM Sans | 主要 UI 字体 | Helvetica Neue, Helvetica, Arial |
| Outfit | 展示标题 | Helvetica Neue, Helvetica, Arial |
| Poppins | 中层标题 | - |
| Roboto | 数据/技术场景 | Helvetica Neue, Helvetica, Arial |

### 4. 组件样式

#### 按钮

**胶囊形主按钮（深色）**
- 背景: `#181e25`
- 文字: `#ffffff`
- 内边距: 11px 20px
- 圆角: 8px
- 用途: 主要 CTA ("开始使用", "了解更多")

**胶囊形导航按钮**
- 背景: `rgba(0, 0, 0, 0.05)`
- 文字: `#18181b`
- 圆角: 9999px (完整胶囊形)
- 用途: 导航标签、筛选切换

**次要浅色按钮**
- 背景: `#f0f0f0`
- 文字: `#333333`
- 圆角: 8px
- 用途: 次要操作

#### 产品卡片

- 背景: 鲜艳渐变（粉/紫/橙/蓝）
- 圆角: 20px–24px
- 阴影: `rgba(44, 30, 116, 0.16) 0px 0px 15px` (品牌紫色光晕)
- 内容: 产品名称、模型版本、描述文字

## 在 PsyClaw 中的应用建议

### 适用场景

1. **聊天机器人界面** (`chatbot.html`)
   - 使用 DM Sans 作为主要字体
   - 按钮使用 8px 圆角
   - 卡片使用品牌紫色阴影

2. **卡片编辑器** (`card.html`)
   - Routine 块使用 20px 圆角
   - 选中状态使用品牌蓝 `#1456f0`
   - 时间轴组件使用品牌色彩体系

3. **整体 UI**
   - 保持白色主导背景
   - 使用紫色色调阴影创造品牌识别度
   - 按钮和导航使用胶囊形或 8px 圆角

### 设计原则

1. **白色优先**: 让产品卡片和插图提供色彩，背景保持纯净
2. **圆润友好**: 避免尖锐边角，使用 20-24px 圆角于产品卡片
3. **品牌一致性**: 紫色阴影连接整个视觉系统
4. **字体分工**: DM Sans 处理 70% 的文本，Outfit 仅用于展示
5. **适中字重**: 标题使用 500-600，700 仅用于正文强调
